Build Powerful Pig Projects

A open source framework for building, organizing, testing, and deploying projects that use Apache Pig. Think "Rails for Pig".

(Bear with us... we're in the process of moving documentation out of help.mortardata.com.)

Organize

The framework provides a standard place for your control scripts, Pig code, UDFs, tests, and libraries. That means people new to your project will be able to understand it quickly. It means it is easier to build reusable code.

Quick Start

After you install the framework, it will install and sandbox all dependencies, including Hadoop and Pig, directly on your workstation.

Plugins

Mortar supports a plugin architecture so you can override or add any functionality you need, without forking the project. Plugins are also great for functionality that is needed by some, but not most, of the community.

An example of a plugin is Watchtower, which is not part of the core project because it has additional dependencies.

Speed Dev

Watchtower is a Mortar plugin that literally watches your Mortar project. It displays the effect of your code, step by step in your browser. As you edit your project (Pig, UDF or local data), Watchtower immediately updates your browser with changes.

Even if you don't install Watchtower, you can use Illustrate for the same purpose, it just doesn't provide live feedback.

Finally, Mortar has begun adding utilities that help you quickly explore new datasets. "Characterize" will recursively traverse a dataset, finding fields, guessing types, counting, keeping common values, calculating deciles and more.

Develop Locally,
Execute Remotely

You can run everything right on your workstation, but when you're ready to run on a real Hadoop cluster, execution and deploy is easy. The framework knows how to ship all your code to your cluster for execution.

Today, the only remote execution target the framework understands is mortardata.com's platform-as-a-service. Help us make the framework universal. Give us a pull request!

Contributing to Mortar

The Mortar Framework is an open source volunteer project initially spun out of mortardata.com. We encourage you to learn about the project and contribute your expertise.
